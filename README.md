<h1>Landing Page — Hotel Booking</h1>

<p>
  Це односторінковий лендінг для готелю, який дозволяє переглядати номери та робити бронювання.
  Проєкт реалізовано з використанням HTML, SCSS, JavaScript та анімаційної бібліотеки GSAP.
</p>

<h2>Демо версія</h2>
<p>
  <a href="https://hotel-booking-room.netlify.app/" target="_blank">Переглянути демо</a>
</p>

<h2>Функціонал проєкту</h2>
<ul>
  <li>Адаптивна верстка для мобільних пристроїв, планшетів та десктопів.</li>
  <li>Плавні анімації елементів під час прокрутки (GSAP + ScrollTrigger).</li>
  <li>Секції з номерами та можливістю бронювання кімнат.</li>
  <li>Кнопка "Повернутися нагору".</li>
  <li>Ховер-ефекти та плавні переходи для елементів інтерфейсу.</li>
</ul>

<h2>Технології</h2>
<ul>
  <li>HTML5</li>
  <li>SCSS (структурований код та адаптивність)</li>
  <li>JavaScript (ES6+)</li>
  <li>GSAP 3 + ScrollTrigger</li>
</ul>

<h2>Особливості</h2>
<ul>
  <li>Реалістична структура лендінгу з секціями: хедер, hero, номери, галерея, футер.</li>
  <li>Анімації оптимізовані для мобільних та планшетів: контент не зникає при прокрутці.</li>
  <li>Код організований за принципами чистої та зрозумілої верстки.</li>
</ul>

<h2>Як використовувати</h2>
<ol>
  <li>Склонуйте репозиторій або завантажте файли.</li>
  <li>Відкрийте файл <strong>index.html</strong> у браузері.</li>
  <li>Сайт працює локально без додаткових налаштувань.</li>
</ol>
